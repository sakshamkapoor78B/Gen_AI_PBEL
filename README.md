# 🚀 45-Day Generative AI Internship

Welcome to my **45-Day Generative AI Internship** repository!

This repository contains all my learning materials, daily practice, projects, assignments, and notes completed during the internship. The goal of this internship is to gain hands-on experience with Generative AI technologies, Large Language Models (LLMs), prompt engineering, AI applications, and modern AI development tools.
